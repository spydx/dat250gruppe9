# iotdevice

A voting "device" for FeedApp

Desined for iOS/Andriod and Web

To run for web
```
> flutter channel master
> flutter upgrade
> flutter config --enable-web
> flutter run -d chrome
```

To run for Android device
```
> flutter run -d android 
```

To run for iOS
```
> flutter run -d ios
```
